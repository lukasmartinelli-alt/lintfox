# Lintcat

Lintcat uses Bash and [Nigit](https://github.com/lukasmartinelli/nigit) to provide a **HTTP API**
to most linters for most programming languages, data formats and configuration files.

## Support

- JavaScript
  - eslint
  - jshint
  - jslint
- Ruby
  - rubocop
  - ruby-lint
- Python
  - flake8
  - pep8
  - pylint
  - pyflakes
- GeoJSON
  - geojsonhint
- JSON
  - jsonlint
- Bash
  - shellcheck
